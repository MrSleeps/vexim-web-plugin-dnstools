# VExim Plugin Autodiscovery

[![Latest Version on Packagist](https://img.shields.io/packagist/v/mrsleeps/vexim-web-plugin-dnstools.svg?style=flat-square)](https://packagist.org/packages/mrsleeps/vexim-web-plugin-dnstools)

This plugin is designed to work with the [VExim Web UI](https://github.com/MrSleeps/VExim-Web-UI).

It provides autodiscovery/autoconfig (autodiscover.your.domain). Helps setting up emails with email clients.

You need to install [VExim Web UI](https://github.com/MrSleeps/VExim-Web-UI) for this to work

[More information about this plugin can be found in the Wiki.](https://github.com/MrSleeps/VExim-Web-UI/wiki)

[Please add any issues to the main VExim Web UI issues page on GitHub.](https://github.com/MrSleeps/VExim-Web-UI/issues)
